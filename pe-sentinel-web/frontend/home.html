What is PE-Sentinel?

PE-Sentinel is a comprehensive static malware analysis tool designed for security researchers, malware analysts, and students learning reverse engineering.

Unlike dynamic analysis (running malware in a sandbox), PE-Sentinel examines the binary structure without execution‚Äîmaking it safe and fast.

Key Capabilities:
  • Structural Analysis: Section entropy, size ratios, permissions
  • Behavioral Analysis: API capability mapping & correlation
  • String Analysis: Extract URLs, IPs, suspicious patterns
  • YARA Scanning: Custom rule support
  • MITRE ATT&CK: Automatic technique mapping
  • Trust Signals: Signature, version info, manifest detection

Core Features

Entropy Analysis

Detect packed/encrypted sections using Shannon entropy. Segment-level analysis catches padding-based evasion.

API Correlation

Map imported functions to malicious capabilities: injection, keylogging, ransomware, credential theft.

MITRE ATT&CK

Automatic mapping to MITRE techniques with confidence scoring and direct links to documentation.

YARA Integration

Run built-in rules or write custom YARA signatures. Full match details with string offsets.

IOC Extraction

Extract URLs, IP addresses, domains, file paths, and registry keys for threat intelligence.

Trust Scoring

Evaluate digital signatures, version info, and manifests. Reduce false positives on legitimate software.
